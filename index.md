1. Purpose
AuditSnap 17025 is a mobile internal audit assistant for ISO/IEC 17025 testing laboratories. It helps laboratories, QA managers, technical managers, internal auditors, certified trainers, and trainees conduct, record, and report internal audits using a structured mobile workflow.
The app supports audit creation, clause-wise audit entry, evidence capture, test witness records, nonconformity management, PDF/Excel report generation, backup, restore, archive, and audit transfer.
________________________________________
2. First-Time Setup
After installing the app, complete the configuration before using it for formal audits.
Open:
Setting / Backup / Config → Configuration
Then complete:
1.	Company Profile

2.	Company Logo

3.	ISO Clause Text Import, if required
________________________________________
3. Company Profile and Logo
Open:
Configuration → Company Profile
Enter the laboratory or organization details such as company name, address, contact details, and report title. Import the company logo if required.
These details are used in PDF and Excel reports.
________________________________________
4. ISO Clause Text Import
AuditSnap 17025 does not bundle ISO/IEC 17025 standard text. Users may import clause text or checklist text from their own authorized source.
Open:
Configuration → Import ISO Clause Text
Use:
1.	Download XLSX Template

2.	Fill clause number, title, and clause text

3.	Import the completed XLSX/CSV file
Imported clause text will be used in audit reports.
________________________________________
5. Creating a New Audit
Open:
New Audit
Enter the audit details, such as audit name, audit date, department, auditor, auditee, scope, and remarks.
After saving, the audit can be opened for clause-wise audit entry.
________________________________________
6. Clause Checklist and Audit Entry
Open the audit and select the required department checklist.
For each applicable clause, record:
1.	Documentation adequacy

2.	Implementation status

3.	Observation / finding

4.	Remark

5.	Evidence checklist

6.	Attachment reference, if applicable
Typical remarks include:
•	Satisfactory

•	Non conformity

•	Reference only

•	Not Applicable

•	Not Assessed
________________________________________
7. Photo and Document Evidence
AuditSnap 17025 supports evidence capture and attachment.
Photo evidence may be captured using the device camera and edited before saving.
Document evidence may be attached using the document evidence function.
Saved evidence can be included in the evidence annexure and audit report.
________________________________________
8. Test Witness
The Test Witness module is used to record witnessed testing activities, including test parameters, current result, retained result, difference, acceptance limit, and remarks.
This is useful when auditors witness laboratory testing during internal audits.
________________________________________
9. Nonconformity Register and Closure
When an audit finding is recorded as a nonconformity, it can be managed through the NC Register.
The NC workflow may include:
1.	NC details

2.	Correction

3.	Root cause

4.	Corrective action

5.	Responsibility

6.	Target date

7.	Closure evidence

8.	Verification and closure status
________________________________________
10. Report Generation
AuditSnap 17025 can generate:
1.	Full Internal Audit Report — PDF

2.	Evidence Annexure — PDF

3.	Post Audit NC Closure Report — PDF

4.	Full Internal Audit Report — Excel

5.	Evidence Annexure — Excel

6.	Post Audit NC Closure Report — Excel
Reports use the configured company profile, logo, audit details, clause entries, evidence, witness records, and NC closure details.
________________________________________
11. Backup and Restore
Open:
Setting / Backup / Config
Use the backup function to create a full audit export package. This package may contain the database, evidence files, report files, and related audit data.
Use:
Restore Full Backup Package ZIP
to restore a previously created backup package.
Important: before restoring, ensure the selected backup file is correct. Restoring a backup may replace the current database.
________________________________________
12. Archive and Audit Transfer
The Archive Manager can be used to move completed or inactive audits into archive and restore them when required.
Audit transfer functions can be used to move or merge audit data between devices, where applicable.
________________________________________
13. Data and Privacy Note
AuditSnap 17025 stores audit data locally on the user’s device. The app does not use a cloud server and does not automatically transmit audit data.
Users may manually export, share, backup, restore, or transfer files using app functions. Users are responsible for protecting exported reports, backup packages, photos, documents, and audit records.
________________________________________
14. Known Limitations
AuditSnap 17025 is a mobile audit assistant and does not replace the responsibility of competent auditors, laboratory management, or authorized quality personnel.
Users must ensure that imported clause text, checklist text, logos, documents, and evidence files are legally authorized for their use.
The app does not automatically verify ISO/IEC 17025 compliance. It supports structured recording, evidence capture, report generation, and audit workflow management.
